Data Overview

Entities

  • Non-profit organizations
  • For-profit organizations
  • Individual artists
  • Unincorporated entities - Fractured Atlas fiscally sponsored projects
  • Government - state and local arts agencies
  • Government - national

Audience

  • Audience lists from local arts organizations
  • Survey for Public Participation in the Arts
  • Cultural Data Project

Demographic & Consumer Behavior

  • U.S. Census (2000 & 2010) - PolicyMap API
  • Consumer behavior - Scarborough

Funding

  • Private grants - Foundation Center API
  • Federal arts grants - NEA
  • State arts grants - NASAA
  • Unincorporated/individual - Kickstarter
  • Individual donors - Vanguard

Spaces

  • Fractured Atlas Spaces API
